## Implementation steps

1. Create `src/main/java/com/coresystem/coresystembackend/controller/AuthUserController.java`, `@RestController @RequestMapping("/api/auth")`.
2. Constructor-inject `LdapUcsService`, `JwtUtils`, `UserRepository` (NO field `@Autowired` вЂ” constitution В§C-004/D-005).
3. `@PostMapping("/dologin") login(@RequestBody LoginRequest req)` (method name `login` вЂ” implemented; the HTTP contract `/api/auth/dologin` is what's externally binding, not the Java method name):
   - `LdapAuthResult r = ldapUcsService.authLDAPNew(req.getUname(), req.getPass())`.
   - If `r == null` в†’ `400 MessageResponse("Failed to connect to LDAP service")`.
   - If `responseCode` is "00" or "01": `String jwt = jwtUtils.generateTokenFromUname(req.getUname())`; `Users usr = userRepository.findByUname(req.getUname()).orElseThrow(...)`; `String urole = "ROLE_" + usr.getUrole()`; return `200 new JwtResponse(jwt, usr.getId(), usr.getUname(), usr.getKodeMitra(), urole)`.
   - Else в†’ `400 MessageResponse("Authentication failed")` (generic вЂ” OQ-FL-3 RESOLVED: do NOT echo raw upstream `responseDescription`, which may contain internal error text; map to generic, log raw detail server-side per В§B-007).
   - Wrap in try/catch в†’ `401 MessageResponse("Authentication failed")` on exception (matches newmojf `AuthUserController.java:140-142`).
4. Do NOT add `@CrossOrigin` (CORS handled by U-006 SecurityConfig).
5. Modify `application.yaml`: add `coresystem.app.jwtSecret: ${JWT_SECRET:dev-only-secret-change-me}` (env-var placeholder вЂ” OQ-AR-3), `coresystem.app.jwtExpirationMs: 86400000`, `spring.datasource.url/username/password` (PostgreSQL `${...}` placeholders pending OQ-AR-2), `spring.jpa.hibernate.ddl-auto: none` (OQ-DM-3), `spring.jpa.show-sql: false`. Keep existing `spring.application.name`.
6. Write `AuthUserControllerTest` (`@WebMvcTest(AuthUserController.class)` with `@MockBean` LdapUcsService/JwtUtils/UserRepository): success (200 + JwtResponse fields `token,type,id,uname,mitKode,urole`), bad-cred (400), null-ldap (400 "Failed to connect"), exception (401).
7. `./mvnw test -Dtest=AuthUserControllerTest`.

- DO NOT put LDAP/JWT/lookup orchestration logic inline as business logic вЂ” delegate to `LdapUcsService`/`JwtUtils`/`UserRepository`; controller only parses + delegates (constitution В§C-003).
- DO NOT use field `@Autowired` (constitution В§C-004/D-005 вЂ” constructor injection).
